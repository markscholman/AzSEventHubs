# Azure Stack Event Hubs

## Introduction

This is the sample code i used for the YouTube video "Azure Stack Hub - Event Hubs (Tenant Experience)

[![Azure Stack Event Hubs](http://img.youtube.com/vi/n4XyMt-kWuI/0.jpg)](http://www.youtube.com/watch?v=n4XyMt-kWuI "Azure Stack Event Hubs")

## How to use

Deploy Azure Stack EventHub on your Azure Stack environment and configure namespace and eventhub. Then create a access policy and use this connection string in this sample Sender and Receiver app.
